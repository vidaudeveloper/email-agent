# GPT Image 2 Prompt Templates

## Product hero (e-commerce)

```
Square 1:1 packshot on pure white seamless background.
Center a pair of matte-black wireless earbuds in open charging case,
soft studio softbox from upper left, subtle ground shadow,
hyper-realistic product photography, sharp micro-texture on plastic,
no visible brand logos except user product shape from reference.
Avoid: fake trademarks, floating artifacts, oversaturated HDR.
```

## Vertical ad poster

```
Design a 9:16 vertical poster.
Exact readable text: "Summer Drop" / "Free shipping today" / "Shop now".
Hero product occupies lower two-thirds, gradient sky background,
clean sans-serif hierarchy, promotional spacing, legible from distance.
Avoid: unreadable microtext, fake sponsor logos.
```

## UGC lifestyle still

```
Vertical 9:16 amateur smartphone photo aesthetic,
product on kitchen counter, natural window side light,
slight handheld imperfection, real-world props: mug, cutting board,
eye-level 28mm feel, unprocessed iPhone quality.
No identifiable human face.
```

## UI mockup

```
1290×2796 smartphone screen, fictional fintech app "AURAE".
Include top header text: "Good morning" / "Balance $12,480.36".
Cards for spending categories, line chart, bottom tab bar icons,
crisp typography, 8pt grid spacing, production-quality mockup.
Palette: indigo accent on white, no real bank branding.
```

## Character / identity board (script2film)

```
16-panel grid character reference sheet, original stylized female presenter,
consistent outfit and hair across panels: front, 3/4, profile, expressions,
neutral studio background, photorealistic, no watermark,
no resemblance to any real celebrity.
```

## Infographic / explainer

```
Landscape 16:9 museum-style disassembly infographic.
Fixed zones: top title, left exploded components, right callout legend.
Exact labels: "Driver unit" / "Battery module" / "ANC mic".
Thin leader lines, muted academic palette, white background, publication-grade.
```
